## MDX Component Examples

This site now supports MDX - Markdown with embedded React components. Here are the available components:

### Callout Blocks

<Callout type="note" title="Note">
Use callouts to highlight important information in your posts.
</Callout>

### Collapsible Sections

<CollapsibleSection title="Example: Algorithm complexity analysis">

Time complexity analysis of the factorial function:

- **Best case**: O(n)
- **Average case**: O(n)

- **Space complexity**: O(n) due to call stack

</CollapsibleSection>

### Interactive Code Blocks

<CodePlayground

## Usage

To use these components in your MDX posts, import them at the top:

```mdx
import { Callout } from '../../components/mdx/Callout';
```

Then use them anywhere in your markdown.
